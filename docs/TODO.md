#### 発見された過払いユーザー
| ユーザー | 過払い額 | 問題 |

| 59C23C | $1,090.84 | HOLDフェーズで払出 |
| 356B74 | $6.40 | 二重払い |

#### 正しいサイクルロジック

**1サイクル = $2,200（$1,100 USDT + $1,100 HOLD → NFT）**

1. **紹介報酬を獲得すると、まずUSDT枠（$1,100）に入る** → 出金できる
2. **USDT枠が$1,100を超えたら、超えた分はHOLD枠に入る** → 出金できない
3. **HOLD枠が$1,100に達したら、NFTが自動購入される** → 余りは次のUSDT枠へ
4. **これを繰り返す**

| 1月 | $549.46 | $549.46 | $0 | - |
| **合計** | $3,026.86 | $1,926.86 | - | 1個 |
